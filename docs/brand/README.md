# Colors of Glory Brand Concepts

This folder stores early visual identity explorations for Colors of Glory.

## Logo Concepts V1

File: `colors-of-glory-logo-concepts-v1.png`

Direction:
- Simple, sleek, app-ready marks for Colors of Glory.
- Inspired by Ezekiel 1:28, interpreted with restraint through aurora, incense, glory, breath, and sound-wave language.
- Warm cream, charcoal, muted gold, and very subtle spectral color accents.
- Avoids generic church symbols, clipart, loud rainbow gradients, and complex illustration.
