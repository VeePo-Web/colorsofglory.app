# Colors of Glory A1 Logo Pack

This package contains the selected A1 direction: a quiet aurora halo with a central flame/eye motif inspired by Ezekiel 1:28. The mark is intentionally simple, warm, and restrained for the Colors of Glory product system.

## Recommended Website Usage

- Header logo: `web-ready/header-logo.png` or `svg/colors-of-glory-a1-lockup-horizontal.svg`
- Small mark / icon button: `web-ready/mark-transparent.png` or `svg/colors-of-glory-a1-mark.svg`
- Favicon: `favicon/favicon.ico` plus `favicon/favicon-32x32.png`
- Apple/mobile icon: `favicon/apple-touch-icon.png`
- Android/PWA icon: `favicon/android-chrome-192x192.png` and `favicon/android-chrome-512x512.png`
- Open Graph/social preview: `social/og-image.png`

## Transparency Notes

The SVG master files and the PNGs in `png/transparent/` use alpha 0 transparent backgrounds. Header lockups are also transparent. The Apple and Android app icons use the warm cream brand tile because many launchers render transparent app icons poorly.

## Color Tokens

- Charcoal: `#1C1A17`
- Cream: `#F5F0E8`
- Gold: `#B8953A`
- Gold light: `#D4AE5C`
- Aurora violet: `#8070C4`
- Aurora blue: `#4D8FD2`
- Aurora green: `#53AB8B`
- Ember: `#E15E39`

## Brand Guardrails

Use this mark with generous whitespace. Do not add music notes, crosses, birds, lens flares, heavy shadows, or extra gradients behind the transparent mark. Let the aurora arc and flame carry the spiritual imagery quietly.
