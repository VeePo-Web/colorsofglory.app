# Handoff — Takes Drawer + Friendly Names

_Owner: Claude. Backend + SDK shipped._

## SDK

```ts
import {
  listTakes, createTake, setPrimaryTake, archiveTake, unarchiveTake,
  renameTake, buildTakeStoragePath, getTakeSignedUrl, type Take,
} from "@/integrations/cog/takes";
```

## Schema notes
- Every voice memo has ≥1 take (backfilled). Exactly one take per memo can be `is_primary` (partial unique index enforces this).
- `is_archived` hides without deleting. There is NO delete UI.
- `friendly_name` is auto-generated by the `set_default_take_name` trigger using `profiles.timezone`. Once a user renames, `name_is_custom = true` and the trigger leaves it alone.
- Format: `"Sunday afternoon · 2 min 14 sec"`. Buckets: morning 5–10, midday 11–13, afternoon 14–16, evening 17–20, late night 21–4.

## Drawer UI
- Component: `src/components/cog/TakesDrawer.tsx`. Trigger from any voice memo card.
- shadcn `Sheet` side=`bottom`, ~60vh.
- Header: memo title + small "Record new take" gold pill (hold-to-record). Holding while primary plays = "Record over this" overlay copy.
- List: takes via `listTakes(voice_memo_id)`. Sort already returned: primary first, then created_at desc.
- Row: 64px tall · waveform sparkline (`waveform_peaks`) · friendly_name (1 line) · duration · primary gold dot · `…` menu.
- Row tap: inline play (uses global mini-player — push to `usePlayerStore`).
- `…` menu items: `Use this take` (→ `setPrimaryTake`), `Rename` (modal → `renameTake`), `Move to section`, `Download`, `Archive` (→ `archiveTake`).
- Swipe-left on row = archive shortcut.
- Footer disclosure "Show archived (N)" → reloads with `include_archived: true`.

## Profile timezone capture
- On first sign-in (or settings page), set `profiles.timezone = Intl.DateTimeFormat().resolvedOptions().timeZone`. Without this, names default to UTC.

## Activity feed copy
- "Parker recorded *Sunday afternoon · 1 min 22 sec*" — italic friendly_name.

## Hard rules
- No "Take 1/2/3" labels anywhere. Always the friendly name.
- No delete action in any menu — archive only, with restore from "Show archived".
- Storage signed URLs via `getTakeSignedUrl(storage_path)` — never embed raw bucket URLs.
